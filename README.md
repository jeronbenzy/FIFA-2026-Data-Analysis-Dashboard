# FIFA World Cup 2026 Player Analytics Dashboard

## Project Overview
This project showcases data cleaning, aggregation, and interactive dashboard design using Microsoft Excel on a massive dataset of over 54,000 player performance rows. This project leverages a large-scale dataset to build an executive-ready sports analytics interface.

## Key Features Built
* **Data Aggregation:** Built a highly optimized Pivot Table grouping tournament goals by country from a 54,600-row ledger.
* **Dynamic Visualizations:** Integrated a polished PivotChart to instantly highlight top-performing teams at a glance.
* **Interactive Slicers:** Deployed a position-based slicer allowing stakeholders to seamlessly drill down into specific player roles (revealing Panama as a major midfield attacking force with 32 goals).
* **Presentation-Ready Design:** Cleaned up the workspace for corporate delivery by hiding standard gridlines, centering data fields, and establishing a professional visual hierarchy.

##  What I Learned
* How to manage and manipulate large-scale datasets in Excel without performance lags.
* The power of using Pivot Tables to compress thousands of rows of raw data into instant, actionable insights.
* Transforming flat charts into dynamic, interactive tools using Excel Slicers to put data control directly into the user's hands.
